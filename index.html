Featured Project: CAPP 30122 Project

Chicago Crime and Education Analysis

This project explores the relationship between education levels and crime rates across Chicago neighborhoods. Using public datasets, we cleaned, merged, and analyzed the data, then built visualizations to better understand patterns across census tracts.

What I worked on:

  • Data cleaning and preparation
  • Exploratory data analysis
  • Altair visualizations including histograms and scatter plots
  • Interpreting patterns between education and crime-related variables

Tools used: Python, Pandas, Altair, GitHub
